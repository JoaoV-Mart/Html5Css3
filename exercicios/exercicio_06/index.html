Principais formatações

Negrito/Destaque

Nesta frase, temos um termo em negrito usando a tag <b></b> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong></strong> (semântica).

Itálico

Nesta frase, temos um termo em itálico usando a tag <i></i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em></em> (semântica).

Texto marcado

Podemos cirar também um texto marcado com a tag <mark></mark>.

E no outro parágrafo temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto pequeno com a tag <small></small>. Para textos grandes usa-se CSS, por se tratar de forma e não de sentido.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considreado com a tag <del></del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins></ins>.

existe também o sublinhado com a tag<u></u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x2+3 usando a tag<sup></sup>.

Texto subescrito

Para inserir coisas do tipo H2O com tag <sub></sub>.
